# CGPA-Calculator
A modern and responsive CGPA Calculator built using HTML, CSS, and JavaScript. This application allows students to calculate their CGPA across 8 semesters, convert CGPA into percentage, predict grades, save records automatically, switch between light/dark themes, and export results .


## 🚀 Features

✅ 8 Semester SGPA Input

✅ Automatic CGPA Calculation

✅ Percentage Conversion

✅ Grade Prediction

✅ Progress Bar

✅ Dark / Light Theme

✅ Local Storage Support

✅ Responsive Design

✅ Glassmorphism UI

---

## 🖥️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- HTML2PDF.js

## 📊 CGPA Formula

CGPA = (SGPA1 + SGPA2 + SGPA3 + SGPA4 + SGPA5 + SGPA6 + SGPA7 + SGPA8) ÷ 8

---

## 📈 Percentage Formula

Percentage = CGPA × 9.5

> Note: Percentage conversion formula may vary depending on your university.

---

## 🏆 Grade Prediction

| CGPA Range | Grade |
|------------|--------|
| 9.0 – 10.0 | O (Outstanding) |
| 8.0 – 8.99 | A+ |
| 7.0 – 7.99 | A |
| 6.0 – 6.99 | B+ |
| 5.0 – 5.99 | B |
| Below 5.0 | C |

---

## 📱 How to Use

1. Enter SGPA for each semester.
2. Click **Calculate**.
3. View:
   - CGPA
   - Percentage
   - Grade
4. Use Dark Mode for a better experience.

---

## 💾 Data Storage

The application automatically saves entered SGPA values using the browser's Local Storage. Your data remains available even after refreshing the page.
